Как разрешить использование JavaScript в браузере
Откройте браузер Chrome на компьютере.
В правом верхнем углу нажмите на значок ...
Последовательно нажмите Безопасность и конфиденциальность, Настройки сайтов, JavaScript.
Выберите Разрешить сайтам использовать JavaScript как режим работы по умолчанию.
